Neste exemplo, foi utilizado um objeto Scanner para ler uma string digitada pelo usuário. 
Em seguida, ocnverte-se a string para um array de caracteres com o método toCharArray() 
e percorre-se metade desse array, trocando os caracteres de posição até que a string esteja 
completamente invertida. Finalmente, cria-se uma nova string a partir do array de caracteres 
invertido e a imprime em tela. Note que, para imprimir a string invertida, 
utiliza-se o construtor de String que recebe um array de caracteres como parâmetro.
