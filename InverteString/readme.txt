Neste exemplo, utilizamos um objeto Scanner para ler uma string digitada pelo usuário. 
Em seguida, convertemos a string para um array de caracteres com o método toCharArray() 
e percorremos metade desse array, trocando os caracteres de posição até que a string esteja 
completamente invertida. Finalmente, criamos uma nova string a partir do array de caracteres 
invertido e a imprimimos em tela. Note que, para imprimir a string invertida, 
utilizamos o construtor de String que recebe um array de caracteres como parâmetro.
