Neste programa, a entrada é obtida através do objeto "Scanner", 
que lê um número inteiro digitado pelo usuário.
Após as informações serem digitadas, entra um loop while, onde 3 variáveis são atualizadas
até que a terceira variável seja maior ou igual a"num".
A condição 'if (c == num)" verifica se o número digitado pelo usuário está presente na sequência Fibonacci. 
Caso contrário o programa imprime uma mensagem informando que o número digitado não pertence a sequência.
